# Telegram-бот: функциональность и команды

## Основные возможности
- Получение пересказа видео с YouTube по ссылке
- Получение дайджеста или пересказа статьи
- Оценка технических статей
- Интерактивное взаимодействие через команды и сообщения

## Поддерживаемые команды

- `/start` — запуск бота, справка по возможностям
- `/source` — информация об источнике контента

## Сценарии использования

1. **Пересказ видео**
   - Отправьте ссылку на YouTube-видео боту
   - Бот извлечёт субтитры, отправит их в LLM и вернёт пересказ

2. **Пересказ статьи**
   - Отправьте ссылку на статью
   - Бот обработает текст и вернёт краткое содержание

3. **Оценка статьи**
   - Отправьте статью или ссылку на статью
   - Бот выполнит оценку по заданным критериям (актуальность, точность, полезность)

## Взаимодействие с пользователем
- Бот поддерживает работу в группах и личных чатах
- Для администраторов доступны расширенные функции (например, публикация в каналы)

## Примеры сообщений
- "Перескажи это видео: https://youtube.com/..."
- "Оцени статью: <текст или ссылка>"

## Ограничения
- Для корректной работы необходим доступ к YouTube и Telegram API
- Некоторые функции доступны только администраторам
