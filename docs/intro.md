# Видео-ретеллинг бот

**Видео-ретеллинг бот** — это Telegram-бот, который автоматически извлекает субтитры из видео на YouTube, а также находит и пересказывает статьи с популярных ресурсов. Бот использует современные LLM (Large Language Models) для генерации пересказов и дайджестов, а также интегрируется с внешними сервисами для расширения функциональности.

## Основные возможности
- Извлечение и пересказ субтитров из YouTube-видео
- Поиск и пересказ статей с популярных ресурсов
- Оценка технических статей по заданным критериям
- Интеграция с Telegram для взаимодействия с пользователями
- Автоматизация поиска и обработки контента через регулярные задачи

## Технологии
- Java, Spring Boot
- Docker, Docker Compose
- Telegram Bot API
- Интеграция с LLM (DeepSeek, OpenAI)
- yt-dlp для загрузки субтитров
