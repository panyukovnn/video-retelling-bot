# Планировщики (Scheduler jobs)

## Назначение
Планировщики отвечают за автоматический поиск, обработку и публикацию контента без участия пользователя.

## Основные задачи

### SourceParsingJob
- Регулярно ищет новые статьи и материалы на популярных ресурсах
- Сохраняет найденный контент в таблицу `contents`
- Создаёт событие на запуск конвейера обработки (в таблице `processing_events`)

### EventProcessingJob
- Регулярно собирает события из таблицы `processing_events` в не-терминальных состояниях
- Передаёт события соответствующим обработчикам для дальнейшей обработки (пересказ, оценка и т.д.)

## Расписание
- Планировщики запускаются с помощью Spring Scheduler (аннотация `@Scheduled`)
- Периодичность и параметры расписания задаются в настройках приложения

## Пример работы
1. SourceParsingJob находит новую статью → сохраняет в БД → создаёт событие
2. EventProcessingJob видит новое событие → запускает обработку через сервисы
